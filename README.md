# SentimentAnalysis

This project investigates the predictive power of social media and news sentiment on stock price movements for five major technology firms (Apple, Amazon, Google, Microsoft, and Tesla) over the period January 2015 to December 2019. We use lexicon-based approaches (VADER, Bag of Words, TextBlob) and pre-trained language models (FinBERT, RoBERTa) into a Bayesian state-space modeling (HD-SURDLM following Shao et al. (2025)) against machine learning benchmarks.
